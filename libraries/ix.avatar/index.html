ix.avatar

Avatar system for Helix.

This system allows you to create custom avatars for your characters.

ix.avatar.RegisterClass(AVATAR, ClassName)

 View source »

Registers an avatar class.

Parameters

  • AVATAR

    (table) The avatar class to register.

  • ClassName

    (string) The name of the avatar class.

Example Usage

ix.avatar.RegisterClass({
    model = "models/player/Group
    modelDefault = "models/player/Group/Group.mdl",
    materials = {
        ["skin"] = {
            slot = 0,
            albedo = "models/player/Group/Group",
            normal = "models/player/Group/Group_n"
        }
    }
}, "group")
